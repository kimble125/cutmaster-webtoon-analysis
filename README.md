# 🚀 Webtoon Growth Analytics: 흥행 DNA 발굴 및 성과 예측 모델

본 프로젝트는 데이터 기반 의사결정을 수행하는 **콘텐츠 그로스 마케터**를 위한 데이터 분석 및 머신러닝(ML) 시뮬레이터 포트폴리오입니다.
웹툰의 '정성적인 작화 요소(조명, 샷, 앵글)'를 정량적인 'DNA 데이터'로 치환하여, 어떤 시각적 특징이 흥행(조회수)을 견인하는지 인과성을 검증하고, 나아가 썸네일을 개선했을 때의 성과 변화를 예측하는 **"What-If 시뮬레이터"**를 구축했습니다.

## 🎯 주요 내용 (Key Features)
1. **데이터 추출 및 파이프라인 (Data Engineering)**
   - AI-HUB 만화/웹툰 데이터를 파싱하여 약 5만 개의 이미지 라벨링 메타데이터 추출.
   - 서비스 플랫폼의 누적 조회수를 누적된 '월(Months)'로 나누어 **'월평균 연재 조회수(Velocity)'**라는 정규화된 타겟 지표 생성.
2. **다중 회귀 기반 흥행 요인 분석 (Analysis)**
   - 로맨스 장르를 타겟으로 다중 회귀분석 및 상관관계 점검 수행.
   - **'역광 조명'**, **'하이 앵글'** 등의 시각적 특성이 조회수를 견인하는 유의미한 상관성을 입증.
3. **What-If 성과 예측 시뮬레이터 (Predictive Modeling)**
   - Random Forest Regressor를 활용해 **R-Squared 0.795**의 성과 예측 모델 구축.
   - 성과 부진 작품(`A 웹툰`)의 썸네일 DNA(역광 추가 등)를 가상으로 변경했을 때 **조회수가 +9.3% 상승할 것**이라는 구체적인 A/B Test 그로스 액션 제안.

> 💡 **상세 포트폴리오 리포트**: 프로젝트의 기획 의도부터 상세 분석 과정 및 한계점(SWOT)은 `docs/portfolio_planning/` 내의 문서에 정리되어 있습니다.
